Les Codes / invitation-code tooling

Chaque code est valide dans le jeu — même somme de contrôle, copiée octet pour octet. Un code n'est pas révocable : c'est un calcul, pas une liste sur un serveur. Traite chaque lot comme des billets imprimés — note à qui ils partent (la colonne CSV existe pour ça). La v1 déplace la validation côté serveur. Codes are checksum math, not a server list — they cannot be revoked. Treat batches like printed tickets; v1 moves redemption server-side.

🤝 Calculateur de mot du jour / duo-code calculator (Serment du Pont)

Deux joueurs n'arrivent pas à prêter serment ? Entre le prénom d'un joueur (tel qu'il apparaît dans SON jeu) et la date : voici le mot que son appareil affiche. Chaque partenaire tape le prénom et le mot de l'autre. Recovers the exact word a player's device shows, using the game's own math.

💾 Sauvegarde perdue / lost save

La vérité, sans détour : la sauvegarde vit dans le navigateur du joueur, clé bq_save_v1 du localStorage, sur son appareil seulement. Effacer les données du navigateur (ou jouer en navigation privée) l'efface. Il n'existe aucune récupération en v0. Offre un nouveau code d'invitation et des excuses honnêtes. Détails : OPERATIONS.md. Saves are device-local localStorage; no recovery exists at v0 — say so honestly.

Le Rythmethe weekly beat

Le Rythme / compressed operator checklist

  1. Chaque jour (2 min) — un œil sur la boîte courriel (formulaires + réponses), un œil sur la mission du jour ci-dessus. Daily glance: inbox + today's mission.
  2. Jeudi 17 h 30 — arriver à Dolan's, cartes d'invitation en poche (« Les Codes »), étoile dorée visible. Arrive early with printed codes.
  3. Jeudi 18 h – 20 h — le Rendez-vous : accueillir, présenter, laisser les Serments du Pont se prêter (« Dépannage » si un code bloque). Host the rendez-vous; troubleshoot oaths.
  4. Jeudi soir — noter la présence réelle dans le carnet. Des chiffres honnêtes, rien d'autre. Record honest attendance numbers.
  5. Vendredi — canoniser la meilleure quête proposée (« Canonisation »), remercier l'auteur par courriel. Canonise one quest, thank its author.
  6. Lundi — vérifier Netlify Forms et que le dernier déploiement GitHub Actions est vert. Check forms + green deploys.
  7. Chaque mois, 20 h 02 — le rituel mensuel (réservé — à définir). Monthly 20h02 placeholder.
  8. Fin de saison — exporter le CSV des codes imprimés, photographier les cartes restantes, écrire la rétro. Season close: CSV export, photo of leftover cards, retro notes.
  9. Le grand cahier de régie complet : OPERATIONS.md (à la racine du dépôt). Full runbook, at the repo root.
